# 本週作業：全班成績

先讓使用者輸入全班人數，接著讓使用者依序輸入全班成績。

1. 無 array 版，課堂上的考試版本，列出全班總平均，並舉例最高分與最低分的一位同學的座號和成績（score1.cpp）
2. array 版，列出全班總平均，並列出最高分與最低分的所有同學的座號和成績（score2.cpp）
